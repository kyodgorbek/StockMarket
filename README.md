# StockMarket 
Android App

During the project I have used following technologies
- MVVM
- Clean Architecture
- Solid Design Principles
- Kotlin Coroutines
- Kotlin Coroutines Flow
- Jetpack Compose
- DI Dagger Hilt
- ViewModel
- Retrofit
- Moshi
- CSV parsing with OpenCSV
- Local caching with Room
- Custom drawing on a canvas using Compose
